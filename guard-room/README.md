# Guard Room

## Objective

This branch ends with the player obtaining the physical keycard required at the final exit.

## Expected Flow

1. Gain access to the guard room or its contents
2. Work through puzzle, clue, or storage-based obstacles
3. Recover the keycard

## Suggested Theme

This branch is a strong fit for:

- Environmental clue hunting
- Light crypto challenges
- Lockbox or storage puzzles
- Guard notes, schedules, or personal effects

## Reward

The output of this branch should be the keycard artifact, whether represented digitally, physically, or as a code that can be redeemed later in the CTF.

## Branch Structure

- `challenge-1`: entry or discovery
- `challenge-2`: puzzle or decryption gate
- `challenge-3`: keycard recovery
