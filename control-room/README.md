# Control Room

## Objective

This branch gives players access to the prison systems that control movement through the facility. Completing this path should unlock the doors needed to reach the final exit area.

## Expected Flow

1. Gain an initial foothold into a control-room related system
2. Pivot or escalate into a more privileged operational environment
3. Use that access to unlock or disable door controls

## Suggested Theme

This branch is the best fit for:

- CCTV or monitoring portal access
- A prison website or internal control interface
- Windows or Active Directory access
- SQL injection or operational-system abuse

## Reward

The reward from this branch should be a durable change in the environment, such as door controls being unlocked or an access route to the final area becoming available.

## Branch Structure

- `challenge-1`: initial foothold
- `challenge-2`: privilege escalation or lateral movement
- `challenge-3`: operational control of doors or related systems
