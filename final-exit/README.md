# Final Exit

## Objective

This folder contains the final challenge at the prison exit.

## Narrative Goal

By the time the player reaches this stage, they should already have:

- Unlocked the relevant facility doors through the control room path
- Recovered the keycard from the guard room path
- Recovered the pincode from the server room path

## Design Intent

The final challenge should confirm that the player completed the entire prison break rather than skipping ahead. It should combine the artifacts recovered from earlier branches and provide a satisfying end to the scenario.

## Suggested Challenge Shape

- Require both the keycard and pincode before the final interaction becomes possible
- Add one final technical or puzzle-based validation step
- Deliver the final flag in a way that feels like a true escape

## Output

This stage should contain the final flag and close out the full CTF narrative.
