# Initial Challenge

## Purpose

This folder contains the opening challenge that gets the player out of the prison cell and into the main CTF flow.

## Narrative Goal

The player begins confined to their cell and must use the first available clues to stage an escape. This challenge should establish the tone of the CTF and clearly transition players into the three-path mid-game structure.

## Design Intent

- Teach players how clues and artifacts will be delivered
- Establish the prison-break theme immediately
- Be approachable enough to start the CTF cleanly, but not trivial
- Produce a clear unlock condition that opens the branching phase

## Suggested Challenge Shape

- A clue hidden in the cell environment
- A smuggled phone, note, or communications angle
- A small puzzle or technical task that unlocks the cell

## Output

Successful completion should provide access to the next stage of the game and clearly signal that the player may now pursue the control room, guard room, and server room paths.
